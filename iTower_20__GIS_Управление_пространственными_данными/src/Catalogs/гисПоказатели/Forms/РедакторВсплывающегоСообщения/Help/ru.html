Форма предназначена для редактирования шаблона всплывающего сообщения, которое будет отображаться при наведении на объект тематической карты.

Шаблон состоит из операндов, возможными значениями которых могут выступать Текст, Значение, Перенос строки. Каждый операнд может иметь свое стилевое оформление - можно задать цвет, размер, выделить жирным или курсивом.

Текст

Простой текст, который подставляется во всплывающее сообщение без изменений

Значение

Во всплывающее сообщение будет подставлено значение реквизита справочника "Объекты карты" или "Показатели"

Перенос строки

Все последующие операнды будут перенесены на новую строку
