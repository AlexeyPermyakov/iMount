Форма предназначена для показа списка справочника «Объекты карты».

В списке выводится:

  • Наименование - наименование объекта карты;
  • Код - код объекта карты в СИСТЕМЕ (присваивается автоматически);
  • Не гео-объект - означает, что данный объект не связан с учетными объектами, не может быть отображен на карте, а может быть использован только как группа при формировании иерархической структуры справочника;
  • Слой - слой карты, к которому относится данный объект карты (выбирается из справочника «Слои карты»).
