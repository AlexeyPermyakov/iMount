Документ предназначен для формирования вручную либо автоматически списка кадастровых документов земельных участков.

 

Поля формы:

•        «Загрузка из xml» - позволяет загрузить данные из файла (xml) с локального носителя. При загрузке предоставляется выбор типа загружаемого документа: кадастровая выписка, кадастровый паспорт и кадастровый план либо автоматическое определение типа документа.

•        «Загруженная xml» - предназначена для обновления кадастрового документа из загруженного xml-файла. Используется, например, когда в табличной части документа были удалены строки, и необходимо заново добавить их в документ, не загружая файл с локального компьютера.

•        «Номер» – порядковый номер кадастрового документа (присваивается автоматически).

•        «Дата» – дата создания кадастрового документа, заполняется автоматически, возможно изменение вручную.

•        «Тип документа» – это тип кадастрового документа, содержащего сведения о кадастровых земельных участках (заполняется автоматически, если загрузка данных выполняется через кнопку «Загрузка из xml», если данные формируются вручную, поле остается не заполненным).

•        «Слой» - слой из справочника «Слои карты», с которым будут связаны загружаемые земельные участки.

•        «Комментарий» – комментарий к данному кадастровому документу. Поле заполняется автоматически при выборе файла с локального компьютера. Возможно заполнение вручную в произвольном порядке.

 

Вкладка «Земельные участки»

Вкладка предназначена для заполнения и редактирования кадастровых земельных участков, входящих в один кадастровый документ.

Участок:

«Создать земельный участок» – дает возможность создать один новый земельный участок (справочник «Кадастровые земельные участки»).

«Обновить земельный участок» – позволяет внести изменения в данные выбранного земельного участка (справочник «Кадастровые земельные участки»).

Групповые действия:

«Заполнить параметры перевода в WGS-84» - предназначена для заполнения параметров перевода координат объектов из местной системы координат в WGS – 84 (справочник «Параметры перевода в WGS»).

«Очисть пустые строки» - дает возможность удалить строки с незаполненными кадастровыми номерами из списка кадастровых земельных участков.

«Подобрать ЗУ» – позволяет подобрать земельные участки из справочника «Кадастровые земельные участки» в соответствии с кадастровым номером в документе, в случае, если данный земельный участок был создан в справочнике, но по каким–либо причинам не был загружен в документ.

«Обновить ЗУ» – позволяет обновить информацию об имеющихся в Системе земельных участках из загруженного файла (xml).

«Создать ЗУ» – позволяет автоматически создать земельные участки в справочнике «Кадастровые земельные участки», если они не были созданы ранее.

«Создать / Обновить ЗУ» – позволяет обновить информацию о земельных участках, уже имеющихся в Системе, а так же создать земельные участки, которых еще нет в Системе.

«Россреестр» - предназначена для поиска и загрузки в Систему кадастровых земельных участков из Публичной кадастровой карты и сведений о них. Однако, информация о местоположении и границах земельных участков, загруженных с Публичной кадастровой карты, будет отсутствовать.

 

«Присоединяемые файлы» - при нажатии открывается новая вкладка, предназначенная для добавления к кадастровому документу файлов с локального компьютера. Для работы с присоединенными файлами используются стандартные инструменты конфигурации.
