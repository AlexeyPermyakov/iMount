Форма предназначена для поиска кадастровых земельных участков на карте по кадастровому номеру.

В поле ввода нужно задать кадастровый номер, он может быть неточным, например, "31:16:127002:4%". После этого нажать кнопку поиска.

Заполнится табличная часть найденных участков. При клике в строке таблицы будет произведено позиционирование на карте.
