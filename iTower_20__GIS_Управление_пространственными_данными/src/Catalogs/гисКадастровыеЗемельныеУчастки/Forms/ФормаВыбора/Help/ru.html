Форма предназначена для выбора элемента из справочника «Кадастровые земельные участки».

В списке выводится:

  • Наименование - наименование кадастрового земельного участка;
  • Код - код кадастрового земельного участка в СИСТЕМЕ (присваивается автоматически);
  • Кадастровый номер - кадастровый номер земельного участка;
  • Категория земель - категория земель, к которой относится данный участок (выбирается из справочника «Категории земель»);
  • Вид использования - вид разрешенного использования, который допустим для данного типа земель (выбирается из справочника «Виды разрешенного использования»).
