Форма предназначена для прямого получения данных по кадастровым земельным участкам с росреестра.

Начало кадастрового квартала - здесь можно указать фрагмент кадастрового номера, чтобы найти нужный кадастровые кварталы (если заранее неизвестно, сколько кадастровых кварталов имеется). Для работы с кнопкой есть подсказка.

Начало кадастрового номера - здесь можно указать фрагмент кадастрового номер, чтобы найти нужные кадастровые номера. Для работы с кнопкой есть подсказка.

В таблицу данные можно внести:

1) вручную

2) из xls-файла с помощью кнопки "Загрузить из Excel"

3) с самого росреестра по шаблону кадастрового номера (кнопка "Обновить" справа от поля "Начало кадастрового номера")

Изначально в табличной части кадастровые номера красные - это означает, что данных по ним нет и они не будут перенесены в форму документа.

Чтобы строки стали белыми и их можно было перенести в документ нужно воспользоваться кнопкой "Получить кадастровые данные".

После первого выполнения возможно, что некоторые строки останутся красными. Оставшиеся можно дозаполнить с помощью такой же кнопки "Получить кадастровые данные" на контекстном меню конкретной строки.

"Удалить существующие ЗУ" - удаляет из таблиы те кадастровые номера, земельные участки по которым уже есть в системе.
