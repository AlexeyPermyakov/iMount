Документ предназначен для ввода данных по объекту карты по нескольким показателям для последующего построения тематических карт.

«Номер» – номер документа (заполняется автоматически).

«Дата» – дата, на которую вводятся значения показателей (заполняется автоматически в соответствии с датой и временем создания документа, либо вручную).

«Объект карты» – объект карты, для которого вводятся значения показателей для построения тематических карт (выбирается из справочника «Объекты карты»).

Поля в табличной части:

«N» – номер по порядку.

«Показатель» – показатель, по которому будут заполняться значения (выбирается из справочника «Показатели»).

«Значение» – значение данного показателя для выбранного объекта карты (заполняется в произвольной форме).
