Справочник предназначен для формирования вручную либо автоматически списка кадастровых земельных участков и сведений о них.

·         «Присоединенные файлы» - позволяет просмотреть список файлов, загруженных в документ.

·         «Загрузить из xml» - позволяет загрузить данные из файла (xml) с локального носителя. При загрузке предоставляется выбор типа загружаемого документа: кадастровая выписка, кадастровый паспорт. Также информацию можно вносить и изменять вручную.

·         Росреестр – позволяет загрузить кадастровые сведения об участке, которые доступные для публичного просмотра.

·         Наименование – наименование кадастрового земельного участка (заполняется в произвольной форме).

·         Код – код кадастрового земельного участка в СИСТЕМЕ (присваивается автоматически, возможно изменение вручную).

·         Слой – слой из справочника «Слои карты», с которым связан загружаемый земельный участок.

·         Объект учета – это объект учетной системы, с которым будет связан земельный участок.

·         Кадастровый номер – кадастровый номер земельного участка (заполняется в произвольной форме).

·         Категория земель – категория земель, к которой относится данный участок (выбирается из справочника «Категории земель»).

·         Вид использования – вид разрешенного использования, который допустим для данного типа земель (выбирается из справочника «Виды разрешенного использования»).

·         Кадастровая стоимость, руб. – комментарий к данному кадастровому земельному участку (заполняется в произвольной форме).

·         Площадь, м2 – площадь кадастрового участка (заполняется в калькуляторе системе, формат – цифры).

Вкладка «Основные»

Вкладка используется для хранения основной информации о земельном участке.

·         Статус – статус кадастровых сведений о земельном участке (выбирается из справочника «Статусы кадастровых участков»).

·         Вид участка – вид кадастрового участка (выбирается из справочника «Виды кадастровых участков»).

·         Предыдущий кадастровый номер – предыдущий кадастровый номер земельного участка (заполняется в произвольной форме).

·         Вид использования по документу – вид использования земельного участка по документам (заполняется в произвольной форме).

·         ОКАТО – код из общероссийского классификатора объектов административно-территориального деления (заполняется в произвольной форме).

·         ОКТМО – код из общероссийского классификатора территорий муниципальных образований (заполняется в произвольной форме).

·         Почтовый адрес кадастрового земельного участка – почтовый адрес участка (вводится вручную в произвольной форме, либо в форму ввода адреса).

 Вкладка «Контуры»

Вкладка предназначена для заполнения списка контуров, входящих в данный кадастровый земельный участок, и их землепользователей. Количество контуров зависит от вида участка

Блок 1

В блоке содержится список контуров, входящих в данный кадастровый земельный участок.

Табличная часть:

o   Учетный номер – учетный номер части многоконтурного земельного участка.

o   Кадастровый номер – кадастровый номер контура (заполняется в произвольной форме).

o   Геометрия в местной системе координат – координаты поворотных точек объекта в кадастровой проекции (заполняется автоматически из кадастрового документа либо вручную).

o   Геометрия в системе координат WGS-84 – координаты поворотных точек объекта в системе координат WGS-84 (заполняется автоматически при пересчете геометрии из местной системы координат в WGS-84 либо вручную).

Блок 2

·         Кадастровый документ – документ, из которого загружен земельный участок (заполняется автоматически из документа «Кадастровые документы», если кадастровый земельный участок был загружен в систему через кадастровый документ).

·         Параметры перевода в WGS-84 - параметры перевода координат объектов из кадастровой проекции в универсальную. (выбирается из справочника «Параметры перевода в WGS-84»).

Геометрия в системе координат WGS-84

·         Редактировать в таблице – позволяет редактировать в таблице координаты поворотных точек объекта в системе координат WGS-84.

·         Редактировать в формате GeoJSON – позволяет редактировать координаты поворотных точек объекта в системе координат WGS-84 в формате JSON.

·         Перевести в WGS-84 – позволяет пересчитать координаты из местной системы координат в СК WGS-84.

Вкладка «Права»

Вкладка предназначена для ввода данных по правам на данный земельный участок.

·         Дата ГРП – дата государственной регистрации права собственности на кадастровый земельный участок (заполняется в произвольной форме либо выбирается из календаря Системы).

·         Номер ГРП – номер государственной регистрации права собственности на кадастровый земельный участок (заполняется в произвольной форме).

·         Вид права – вид права на земельный участок (выбирается из справочника «Виды прав на кадастровый участок»).

·         Числитель дроби доли / Знаменатель дроби доли – доля участка, на которую распространяется указанное право (заполняется в произвольной форме).

·         Площадь доли – площадь доли участка, на которую распространяется указанное право (заполняется автоматически при загрузке xml-файла либо рассчитывается автоматически при заполнении полей «Числитель дроби доли / Знаменатель дроби доли»).

·         Правообладатель – владелец данного права на участок.

·         Комментарий – комментарий к данным о правах кадастрового участка (заполняется в произвольной форме).

·         Документы - документ подтверждающий наличие права на кадастровый земельный участок (заполняется автоматически при загрузке xml-файла).
