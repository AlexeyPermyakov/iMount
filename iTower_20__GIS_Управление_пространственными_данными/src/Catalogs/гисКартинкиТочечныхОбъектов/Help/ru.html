Справочник предназначен для ведения списка картинок, используемых для отображения на карте объектов точечных слоев.

 

Картинка загружается в Систему с локального компьютера.

В карточке картинки в меню «Картинка» имеются инструменты для загрузки картинки с локального компьютера в Систему и для удаления картинки из Системы.

Для отображения точечных объектов рекомендуется использовать небольшие картинки – иконки или пиктограммы.
