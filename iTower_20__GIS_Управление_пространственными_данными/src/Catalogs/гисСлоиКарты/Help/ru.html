Справочник предназначен для ведения характеристик слоев, включенных в СИСТЕМУ (векторных, растровых, с общедоступных ресурсов, сформированных из учетных объектов Системы и т.д.).

 

При создании нового слоя обязательным для заполнения реквизитом является «Тип слоя».

«Тип слоя» представляет собой группировки и виды для отображения геопространственных объектов:

1)       по справочнику «Объекты карты».

Слой будет сформирован из объектов карты, связанных с учетными объектами системы. При выборе данного типа слоя, появляется дополнительный блок Объект учета, содержащий поле «Справочник». В этом поле указывается справочник учетной системы, объекты которого будут связаны с объектами карты. Если у выбранного справочника имеются владельцы – можно указать конкретного владельца, чтобы отбирать по нему остальные элементы.

«Всплывающее сообщение» - при нажатии открывается форма ввода шаблона всплывающего сообщения, которое будет появляться при наведении мышкой на объект слоя на карте.

«Тип геометрии» - тип геометрии объектов слоя. Выбирается из выпадающего списка: полигон, линия или точка.

«Картинка» - поле доступно только, если тип геометрии слоя – точка, и предназначено для выбора картинки, которая будет отображаться на карте для объектов данного слоя (выбирается из справочника «Картинки точечных объектов»).

«Заливка», «Границы», «Подписи» - блоки предназначены для настройки отображения объектов слоя на карте.

2)       по справочнику «Кадастровые земельные участки».

Слой будет сформирован из объектов, хранящихся в справочнике «Кадастровые земельные участки». Они могут быть связаны с учетными объектами системы. При выборе данного типа слоя, появляется дополнительный блок Объект учета, содержащий поле «Справочник». В этом поле указывается справочник учетной системы, объекты которого будут связаны с объектами карты. Если у выбранного справочника имеются владельцы – можно указать конкретного владельца, чтобы отбирать по нему остальные элементы.

«Всплывающее сообщение» - при нажатии открывается форма ввода шаблона всплывающего сообщения, которое будет появляться при наведении мышкой на объект слоя на карте.

«Заливка», «Границы», «Подписи» - блоки предназначены для настройки отображения объектов слоя на карте.

3)       из файла.

Позволяет подключить в Систему слой с локального компьютера. При выборе этого типа слоя, появляется дополнительное поле «Формат геоданных», где указывается формат загружаемого файла слоя - KML. Кроме того в форме появляются инструменты для загрузки, выгрузки и редактирования данного слоя.

4)       Росреестр – позволяет подключить в Систему публичную кадастровую карту Росреестра.

5)       WMS – позволяет подключить в систему слой (векторный или растровый) с указанного сервера по протоколу WMS.

При выборе этого типа слоя появляются дополнительные поля: «Адрес ГИС-сервера», где указывается ссылка на сервер (заполняется в произвольной форме), и «Имя слоя на ГИС-сервере» - имя слоя на указанном сервере (выбирается из списка доступных на указанном сервере слоев).

«Порядок отображения» - порядок отображения слоя на карте. Чем меньше число, тем ниже будет располагаться слой по отношению к другим слоям.
