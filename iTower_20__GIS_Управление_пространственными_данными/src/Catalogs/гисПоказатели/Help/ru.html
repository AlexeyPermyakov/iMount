Справочник предназначен для ведения показателей и их характеристик для построения тематических карт, а так же для настройки подписей тематики и шаблонов всплывающих сообщений.

 

«Настройка показателя для слоя» - в данном поле отображается наименование слоя, для которого осуществляется настройка показателя. Наименование слоя отображается в соответствии с выбранным слоем в Блоке 1.

«Всплывающее сообщение» - поле предназначено для заполнения шаблона всплывающего сообщения, которое будет отображаться при наведении на объект тематической карты. Шаблон всплывающего сообщения отображается в соответствии с выбранным слоем в Блоке 1.

«Наименование» – наименование показателя (заполняется в произвольной форме).

«Код» – код показателя (присваивается автоматически).

«Группа» – ссылка на групповой элемент справочника. Таким образом формируется иерархическая структура справочника (выбирается из справочника «Показатели»).

«Тип значения» – тип значения показателя (выбирается из Системы из выпадающего списка: число, булево, справочник).

«Справочник» - выбор справочника, из которого заполняется значение показателя (актуально, только если тип значения показателя «Справочник»).

«Количество дробных знаков» - количество знаков после запятой (актуально, только если тип значения показателя «Число»).

«Периодичность» – частота внесения данных. Выбирается из выпадающего списка: год, квартал, месяц.

 

Блок 1 предназначен для формирования слоев, по которым может быть сформирована тематическая карта по выбранному показателю.

Поля табличной части:

«N» - номер слоя по порядку (заполняется автоматически при добавлении слоя).

«Слой карты» - слой, по которому может быть сформирована тематическая карта по выбранному показателю (выбирается из справочника «Слои карты»).

«Всплывающее сообщение» - поле предназначено для заполнения шаблона всплывающего сообщения, которое будет отображаться при наведении на объект тематической карты.

 

Блок 2 предназначен для настройки легенды по показателю. Если в Блоке 1 указано несколько слоев, легенда настраивается для каждого слоя отдельно.

«Авто» – инструмент предназначен для автоматического создания цветовой шкалы, для этого нужно указать цвет начала и окончания легенды, мин и макс значение показателя и количество градаций.

«Заполнить подписи» – позволяет автоматически заполнить подписи для всех строк табличной части в соответствии с указанными интервалами значений показателя.

Поля табличной части:

«N» – номер по порядку.

«Цвет» – цвет, в который будут открашиваться значения показателя в пределах заданного интервала (в соответствии с автоматическим режимом либо выбирается вручную).

«Значение от» / «Значение до» – минимальное / максимальное значение диапазона (заполняется автоматически, при автоматическом режиме, либо значения добавляются вручную для типа значения показателя - число).

«Значение» – значение показателя (заполняется автоматически, при автоматическом режиме, либо значения добавляются вручную для типа значения показателя: булево, справочник).

«Подпись» – комментарий к цвету (заполняется в произвольной форме либо автоматически при использовании инструмента «Заполнить подписи»).

 

«Значения показателей объектов» - при нажатии открывается новая вкладка, предназначенная для просмотра значений показателей. Значения показателей по объектам карты вносятся через документ «Ввод значений показателей по объекту» или «Ввод значений показателей по списку  объектов».
