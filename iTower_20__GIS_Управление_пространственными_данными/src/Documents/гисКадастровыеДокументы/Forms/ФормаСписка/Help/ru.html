Форма предназначена для показа списка документов «Кадастровые документы».

В списке выводится:

  • Дата - дата создания документа;
  • Номер - присваивается автоматически;
  • Тип документа - указывает автоматически;
  • Комментарий - автоматически сюда записывается имя файла, из которого была сделана загрузка.
