Форма предназначена для редактирования параметров перевода координат объектов из кадастровой проекции в универсальную проекцию координат.

 

«Наименование» – наименование набора параметров перевода из кадастровой проекции (заполняется в произвольной форме).

«Код» – код параметра перевода (присваивается автоматически).

«Алгоритм перевода» – метод пересчета координат объекта по заданным параметрам. В случае применения в качестве алгоритма перевода ГОСТа-2008 используются геоцентрические преобразования, внесенные в Систему в качестве констант, и закрытые для редактирования пользователем. В случае если ГОСТ-2008 не подходят для перевода координат, пользователь может использовать другой алгоритм перевода. При этом на вкладке «Дополнительно» (вкладка появляется, когда снята отметка «Алгоритм перевода ГОСТ-2008») пользователь должен заполнить геоцентрические преобразования, которые будут использоваться при переводе координат данным методом.

 

Вкладка «Основные»:

«Осевой меридиан» – долгота осевого меридиана, проходящего через начало местной системы координат (заполняется в произвольной форме).

«Смещение на восток» – заполняется в произвольной форме.

«Смещение на север» – заполняется в произвольной форме.

В нижней части формы представлены поля для внесения значений поправок к координатам. При пересчете координат из МСК (СК) в WGS все поправки должны быть положительными.

 

Вкладка «Дополнительно»:

Вкладка активна только, если снята отметка алгоритм ГОСТ-2008, и предназначена для заполнения геоцентрических преобразований, которые будут использоваться при переводе координат данным методом.
