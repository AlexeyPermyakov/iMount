Справочник хранит список объектов, которые могут быть визуализированы на карте.

 

При создании нового объекта карты необходимо сначала создать слой, элементом которого будет являться и свойства которого примет создаваемый объект (справочник «Слои карты»).

От настроек выбранного слоя будет зависеть вид формы «Объекта карты».

Если тип геометрии слоя – точка, то в форме объекта карты будут добавлены поля для ввода координат (долгота, широта). Если тип геометрии слоя – линия или полигон, то в форме объекта карты появляются поля для ввода в таблицу и для ввода в формате JSON координат поворотных точек объекта в системе координат WGS-84.

Отметка «Не гео-объект» означает, что данный объект не связан с учетными объектами, не может быть отображен на карте, а может быть использован только как группа при формировании иерархической структуры справочника.

«Объект учета» - это объект учетной системы, с которым связан объект карты.

 

«Значения показателей объектов» - при нажатии открывается новая вкладка, предназначенная для просмотра значений показателей. Значения показателей по объектам карты вносятся через документ «Ввод значений показателей по объекту» или «Ввод значений показателей по списку объектов».

 

«Присоединяемые файлы» - при нажатии открывается новая вкладка, предназначенная для добавления к объекту учета файлов с локального компьютера. Для работы с присоединенными файлами используются стандартные инструменты конфигурации.
